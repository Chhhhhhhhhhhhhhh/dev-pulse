# ⚡ Dev-Pulse

**开发者生产力仪表盘** — 你深度工作的指挥中心。

追踪专注时长、管理任务、同步 GitHub 动态、生成周报 — 一切尽在一台精工仪表盘中。

## 功能特性

- **番茄钟** — 25 分钟 / 50 分钟专注时段、5 分钟 / 15 分钟休息，带音频提示和桌面通知
- **任务管理** — 四级优先级（紧急/高/中/低）、状态筛选、类 GitHub 复选框交互
- **GitHub 集成** — 查看 Issue、PR、通知和动态事件
- **周报导出** — 一键导出 Markdown 周度总结
- **标签系统** — 为每个专注时段打标签（写代码、开会、学习、修 Bug、写文档、Code Review、规划）
- **国际化** — 中英文界面切换
- **暗色/亮色主题** — 深色优先的工业美学

## 快速开始

```bash
# 安装依赖
npm install

# 启动服务
npm start
```

浏览器打开 `http://localhost:3855` 即可使用。

## 技术栈

- **后端**：Node.js + Express
- **数据库**：SQLite（better-sqlite3）
- **前端**：原生 HTML/CSS/JS — 零框架、零构建步骤
- **设计**：OKLCH 色彩空间 · Space Grotesk + JetBrains Mono · 粗野/工业实用主义风格
