## Design Context

### Users
Developers tracking their own productivity. They use dev-pulse daily — starting focus timers, checking tasks, reviewing GitHub activity. Context: solo developer or small team, using it alongside actual coding work.

### Brand Personality
- **Precise**: A tool, not a product. Does one thing well.
- **Focused**: Eliminates distraction. Everything on screen earns its place.
- **Quiet**: No decoration. Form follows function, nothing more.

3-word persona: Sharp · Restrained · Functional

### Aesthetic Direction
**Refined tool aesthetic** — the visual quality of a well-made instrument.

References: Linear (confidence in spacing), Arc browser (restrained chrome), iA Writer (focus).
Anti-references: Generic SaaS dashboards with glass morphism, purple-blue gradients, decorative glow.

Dark-first — this is a tool used during deep work. Light theme available as secondary option.

### Design Principles
1. **Fonts carry meaning**: system-ui for body (native, invisible), Space Grotesk for headings (geometric confidence), JetBrains Mono for data (precision)
2. **Color is earned**: single accent (teal), applied only where it signals state. Everything else is tinted monochrome
3. **Borders over shadows**: 1px hairline borders communicate structure. No floating-card clichés
4. **Space creates hierarchy**: tight grouping for related items, generous separation between sections
5. **Motion means state**: only animate when it tells you what changed. No page-load animations

### Technical
- OKLCH color space throughout
- Fixed rem type scale (1.25 ratio)
- Font: Space Grotesk (headings), system-ui (body), JetBrains Mono (data)
- Top navigation bar — compact, always visible
- Centered content (960px max) with generous padding
- Solid surfaces, no glass morphism
