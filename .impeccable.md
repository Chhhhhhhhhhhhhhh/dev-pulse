## Design Context

### Users
Developers tracking their own productivity. They use dev-pulse daily — starting focus timers, checking tasks, reviewing GitHub activity. Context: solo developer or small team, using it alongside actual coding work.

### Brand Personality
- **Industrial**: Precision tool, not a SaaS product. Trusted instrument.
- **Focused**: Eliminates distraction. Everything on screen has a reason to be there.
- **Confident**: No decoration-for-decoration's-sake. Form follows function.

3-word persona: Sharp · Restrained · Functional

### Aesthetic Direction
**Brüt / Industrial Utilitarian** — like a well-made analog gauge in a digital world.

References: Linear (task management, confidence in spacing), Arc browser (restrained chrome), vintage audio equipment (tactile precision).
Anti-references: Generic SaaS dashboards with glass morphism cards, purple-blue gradients, Inter/Roboto everywhere.

Dark mode only — this is a tool used during deep work, not a marketing page.

### Design Principles
1. **Fonts carry meaning**: system-ui for body (native, invisible), Space Grotesk for headings (geometric confidence), JetBrains Mono for data (precision)
2. **Color is earned**: single accent (teal-green), applied only where it means something. Everything else is tinted monochrome.
3. **Borders over shadows**: 1px hairline borders communicate structure without the "floating card" SaaS cliché
4. **Space creates hierarchy**: tight grouping for related items, generous separation between sections. No card-nesting.
5. **Motion means state**: only animate when it tells you what changed. Ease-out-quart deceleration for natural feel.

### Technical
- OKLCH color space throughout
- Fixed rem type scale (1.25 ratio)
- Font: Space Grotesk (headings), system-ui (body), JetBrains Mono (data)
- Dark-only theme with elevated surface layers
