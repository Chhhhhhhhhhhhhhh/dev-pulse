# ⚡ Dev-Pulse

[English](./README.md) | [中文](./README_zh.md)

**Developer Productivity Dashboard** — your command center for deep work.

Track focus sessions, manage tasks, sync GitHub activity, and generate weekly reports — all from one industrial-grade dashboard.

## Features

- **Pomodoro Timer** — 25m / 50m focus sessions, 5m / 15m breaks, with audio chime & desktop notifications
- **Task Management** — Priority levels (urgent/high/medium/low), status filtering, GitHub-style checkboxes
- **GitHub Integration** — View issues, PRs, notifications, and activity events
- **Weekly Reports** — Export Markdown weekly summaries
- **Tag System** — Categorize sessions (coding, meeting, debugging, learning, writing, reviewing, planning)
- **i18n** — English / 中文 language toggle
- **Dark/Light Theme** — Dark-first industrial aesthetic

## Quick Start

```bash
# Install dependencies
npm install

# Start the server
npm start
```

Open `http://localhost:3855` in your browser.

## Tech Stack

- **Backend**: Node.js + Express
- **Database**: SQLite (better-sqlite3)
- **Frontend**: Vanilla HTML/CSS/JS — zero framework, zero build step
- **Design**: OKLCH color space · Space Grotesk + JetBrains Mono · Brüt/Industrial Utilitarian
